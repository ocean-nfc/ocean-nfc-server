Running the server

Install dependencies

npm install

Create .env file for development mode

Create a file called .env. Copy and paste the contents of .env.example for development mode configuration. This will cause the server to use sqlite instead of postgres (for local development).

Run the server

npm run start

Run tests

npm run test
